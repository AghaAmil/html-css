2. Responsive Image Gallery

Resize the browser window to see the effect.

This example use media queries to re-arrange the images on different screen sizes: for screens larger than 1100px wide, it will show four images side by side, for screens smaller than 600px, it will show two images side by side. For screens smaller than 500px, the images will stack vertically (100%).

You will learn more about media queries and responsive web design later in our CSS Tutorial.
